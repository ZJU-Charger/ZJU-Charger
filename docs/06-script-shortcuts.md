# Script 快捷指令文档

本文档介绍如何使用 iOS 快捷指令快速查询充电桩状态。

## 功能特性

- 🚀 **快速查询**：一键查询充电桩状态
- ❤️ **关注列表**：管理关注的站点
- 📱 **iOS 原生**：使用 iOS 快捷指令，无需安装额外应用

| 快捷指令     | 说明  | 链接       |
| ------------ | ----- | ---------- |
| 查看关注列表 | 查看当前已关注的站点列表           | [查看关注列表.shortcut](https://www.icloud.com/shortcuts/1545aeee457046dbacba42ef0ab6285d) |
| 添加关注     | 添加站点到关注列表                 |                                                                                            |
| 移除关注     | 从关注列表移除站点                 |                                                                                            |
| 查看空闲站点 | 筛选并显示有空闲充电桩的站点       |                                                                                            |
| 查询所有站点 | 实时查询所有充电站点状态           |                                                                                            |
| 查询关注列表 | 查询已关注站点的状态               |                                                                                            |
| 查看缓存数据 | 快速查看缓存数据（不触发实时查询） |                                                                                            |

## 使用方法

1. 点击链接安装对应的快捷指令。
2. 修改快捷指令中的列表，每行一个你关注的站点 ID，注意不要有空行。（ID 可以从尼普顿的充电历史当中找到，一般是一个 8 位数字，比如 `60459109`）。
   > **注意**：快捷指令编辑界面截图暂未提供，请参考上述说明进行操作。
3. 保存快捷指令，并运行。
4. 快捷指令会自动查询关注站点的状态并显示结果。
5. 还可以通过 分享 - 添加到主屏幕的方式，将快捷指令添加到主屏幕，方便随时查看。
   > **注意**：主屏幕图标示例暂未提供，请自行在 iOS 设置中添加。
6. 注意由于服务器 API 访问速率有限制，短时间反复查询会导致请求失败，请勿反复查询。

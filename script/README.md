# iOS 快捷指令使用说明

本目录包含用于访问 ZJU Charger API 的 iOS 快捷指令。

| 快捷指令     | 说明                               | 链接                                                                                       |
| ------------ | ---------------------------------- | ------------------------------------------------------------------------------------------ |
| 查看关注列表 | 查看当前已关注的站点列表           | [查看关注列表.shortcut](https://www.icloud.com/shortcuts/777f5e4279354d31b2f344830d6aa178) |
| 添加关注     | 添加站点到关注列表                 |                                                                                            |
| 移除关注     | 从关注列表移除站点                 |                                                                                            |
| 查看空闲站点 | 筛选并显示有空闲充电桩的站点       |                                                                                            |
| 查询所有站点 | 实时查询所有充电站点状态           |                                                                                            |
| 查询关注列表 | 查询已关注站点的状态               |                                                                                            |
| 查看缓存数据 | 快速查看缓存数据（不触发实时查询） |                                                                                            |
